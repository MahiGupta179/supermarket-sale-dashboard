# 🌟 Overview

The **Global Superstore Sales Dashboard** is a visually appealing and fully interactive **Microsoft Excel Dashboard** built to analyze business performance across multiple dimensions including **Sales, Profit, Shipping Cost, Discounts, Markets, Categories, Customer Segments, and Shipping Modes**.

Using **Pivot Tables**, **Pivot Charts**, **Slicers**, and **Dynamic KPI Cards**, this dashboard enables quick analysis and supports better business decision-making.
